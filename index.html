CSS Checkup

Make a new HTML& CSS page using: class, id, margin and or padding, font, background, width or height, border and anything else which comes to mind.



class

class refers to an element on an HTML page which has been used multiple times. In CSS, calling up class with its octothrope enables stylistic features to be set for all elements within that class.


id

id refers to an element on an HTML page which has specific, unique styling attached to it. In CSS, calling up an id occurs with [.] in the CSS page, thereby allowing these specific changes to be given to that elment. These even override class--and this paragraph is an instance of the id overriding the class. Notice this font is different than what is seen in the paragraphs preceding and following this one.


margin or padding

margin and padding refer to spaces between elements and between the content of an element and its border (even if that border isn't visible). margin creates space around an element, outside defined borders. padding creates space inside an element's defined borders. Both elements have been adjusted via CSS.


font

font and font-family describe the ways in which letters show up, and can be set to bold or other styling along with font-size in a direct way. line-height>also gives the page designer pixel-level control over the visual elements of a text.


background

background refers to the color upon which other elements sit on a page as a whole as well as the color within a shape.
